TALK-AND-ALL
Talk-And-All is a Full Stack Chatting App. Uses Socket.io for real time communication and stores user details in encrypted format in Mongo DB Database.
                                                         
TACH STACK                                                      
Client: ReactJS
Server: NodeJS ExpressJS
Database: MongoDB
